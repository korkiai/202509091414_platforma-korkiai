# Lista Pluginów Platformy KorkiAI

**Wersja**: Pro (płatna)
**Funkcjonalności**:

- Integracja z użytkownikami WordPress

## Konfiguracja

### Wymagania Serwera
- **PHP**: 7.4+ (zalecane 8.0+)
- **MySQL**: 5.7+ (zalecane 8.0+)
- **WordPress**: 6.0+
- **Pamięć**: 256MB+ (zalecane 512MB+)
- **Limit czasu**: 300s+
